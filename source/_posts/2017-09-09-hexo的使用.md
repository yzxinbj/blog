### 写作
- `hexo new "文件名"`
- `hexo generate` 可以追加`-w`参数实时变动
- `hexo deploy` 发布

### 遇到的问题
`deploy` 到`git` 上样式，js 404
```
配置要正确
deploy:
  type: git
  repo: git@github.com:yzxinbj/blog.git
root: /blog  # 默认是斜杠根目录，git 要配置为你的仓库路径

```
