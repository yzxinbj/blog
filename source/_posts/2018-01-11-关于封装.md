## 对象封装
什么是对象？
> 对象：无序属性的集合，其属性可以包含基本值，对象或者函数


创建对象的方法有多少种？

- var obj = {}; // 字面量
- var obj = new Object(); // new


封装对象的目的是什么？
- 隐藏细节实现，开放接口调用
- 复用，使逻辑清晰，代码好维护


怎么封装？

- 原型链?
- 继承?
- new?
```
function Base(name, age){
    this.name = name;
    this.age = age;
}

var obj = new Base();

// 等价于
var obj = {};
obj.__proto__ = Base.prototype;
Base.call(obj);

```
